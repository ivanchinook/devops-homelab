DevOps Home Lab

Overview

This repository represents a hands-on DevOps home lab project focused on CI/CD, containerization, Kubernetes deployment and infrastructure automation.

The goal of the project is to demonstrate real-world DevOps practices rather than theoretical knowledge, with emphasis on:

- automation
- repeatability
- troubleschooting
- production-like workflows
